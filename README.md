Guide
=====

Linux installation
------------------

Backup .vim and .vimrc and execute the following commands:

    # clone vimrc
    cd ~
    git clone https://github.com/chubiei/vimrc
   
    # link .vim and .vimrc
    ln -sf vimrc ~/.vim
    ln -sf vimrc/vimrc ~/.vimrc
   
    # install
    cd ~/.vim
    ./install
